# Лабораторная работа 5

## Тема

Разработка вакансии для компании в сфере онлайн-обучения.

## 1. Состав команды для ближайших проектов

Компания занимается разработкой, продажей и внедрением решений в сфере онлайн-обучения. Для запуска и поддержки новых проектов целесообразно сформировать следующую команду:

1. Product Manager
2. Project Manager
3. UX/UI Designer
4. Frontend Developer
5. Backend Developer
6. Full-stack Developer
7. QA Engineer
8. DevOps Engineer
9. Business Analyst
10. Методист / контент-менеджер образовательной платформы

Если проект небольшой, роли 2-3 специалистов могут совмещаться.

## 2. Требования к отбору кандидатов на должность Web-разработчика

### Цель должности

Разработка и развитие веб-платформы онлайн-обучения: пользовательский интерфейс, личные кабинеты, интеграции, серверная логика, аналитические модули, системы тестирования и обратной связи.

### Основные обязанности

- разработка и доработка веб-интерфейсов платформы;
- реализация пользовательских сценариев;
- участие в проектировании архитектуры модулей;
- интеграция frontend и backend-части;
- работа с API и базами данных;
- исправление ошибок и оптимизация производительности;
- участие в code review и командном взаимодействии;
- документирование принятых решений.

### Требования к кандидату

- высшее образование или обучение по направлению IT;
- понимание жизненного цикла веб-приложения;
- знание HTML, CSS, JavaScript;
- опыт работы с одним из современных frontend-фреймворков;
- понимание клиент-серверной архитектуры;
- базовые навыки работы с backend и REST API;
- знание Git;
- умение читать техническую документацию;
- внимательность к деталям и ответственность;
- умение работать в команде.

### Будет плюсом

- опыт разработки образовательных сервисов;
- знание TypeScript;
- опыт работы с Node.js, Python или PHP;
- понимание CI/CD и Docker;
- наличие pet-проектов или портфолио на GitHub.

## 3. Готовый текст вакансии

## Web-разработчик в EdTech-компанию

Мы создаем цифровые решения для онлайн-обучения: личные кабинеты, курсовые платформы, системы тестирования, трекинг успеваемости и аналитические модули. Ищем web-разработчика, который поможет развивать продукт и делать его удобным для преподавателей, студентов и корпоративных клиентов.

### Чем предстоит заниматься

- разрабатывать и дорабатывать веб-модули платформы;
- реализовывать новый функционал для пользователей;
- поддерживать и улучшать интерфейсы личных кабинетов;
- работать с API, формами, интеграциями и серверной логикой;
- участвовать в обсуждении технических решений;
- исправлять баги и оптимизировать производительность.

### Что мы ожидаем

- уверенные знания HTML, CSS, JavaScript;
- опыт с React и TypeScript;
- понимание принципов адаптивной верстки;
- опыт работы с современными frontend-инструментами;
- базовое понимание backend-разработки и REST API;
- умение работать с Git;
- ответственность, аккуратность, обучаемость;
- готовность работать в команде и обсуждать решения.

### Будет преимуществом

- опыт с Vue/Angular;
- знание Node.js / Python / PHP;
- опыт работы с образовательными платформами;

### Мы предлагаем

- работу над реальными EdTech-продуктами;
- гибкий формат взаимодействия;
- возможность профессионального роста;
- участие в развитии востребованного цифрового продукта;
- сильную проектную команду и понятные задачи.

Вакансия ориентирована на web-разработчика, который умеет сочетать техническую базу, готовность развиваться и интерес к образовательным продуктам. Такой профиль хорошо соответствует задачам компании в сфере онлайн-обучения.
