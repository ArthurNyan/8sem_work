# Лабораторные работы 1-2

## Тема

Анализ зарубежного и отечественного рынка информационных систем для управления и оценки персонала.

## 2. ИТ-обеспечение ассесмент-центров

Ассесмент-центр требует одновременно: массового сбора данных, оценки компетенций, сопоставления результатов и формирования отчетов. Рассмотренные системы поддерживают эти задачи в разной степени.

| Блок ассесмент-центра | Что нужно на практике | Наиболее подходящие системы |
|---|---|---|
| Предварительный отбор | воронки, коммуникации, база кандидатов | Greenhouse, Talantix, Huntflow, Skillaz |
| Комплексная оценка | профили компетенций, performance, матрицы | SAP SuccessFactors, Workday, Mirapolis, PeopleForce |
| Аналитика и отчетность | сравнение кандидатов, HR-метрики, дашборды | Workday, SAP SuccessFactors, Mirapolis |
| Сопровождение после найма | адаптация, развитие, обучение | Mirapolis, SAP SuccessFactors, Workday, PeopleForce |

### Вывод по ассесмент-центрам

Для ассесмент-центров наиболее эффективна комбинированная модель:
- ATS-система для организации потока кандидатов;
- HCM/HRM-система для оценки компетенций, аналитики и последующего развития сотрудников.

## 3. Сопоставление групп решений

| Критерий | Зарубежные решения | Отечественные решения |
|---|---|---|
| Масштаб HCM-функций | очень высокий | от среднего до высокого |
| Локализация под РФ | ограниченная | высокая |
| Стоимость внедрения | чаще высокая | чаще гибкая |
| Поддержка на российском рынке | ограниченная | сильная |
| Готовность к массовому подбору | зависит от системы | обычно высокая |
| Импортонезависимость | низкая | высокая |
