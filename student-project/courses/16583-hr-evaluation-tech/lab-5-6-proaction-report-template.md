многозадачности.

## 6. Условия и мероприятия для прокачки качеств с недостаточным весом

### 6.1. Стрессоустойчивость (620 гр)

- внедрить правило: сначала фиксация проблемы, затем план действий (без импульсивных решений);
- использовать короткие паузы восстановления в течение дня;
- раз в неделю проводить ретроспективу стрессовых ситуаций: причина -> реакция -> улучшение.

### 6.2. Тайм-менеджмент (680 гр)

- планирование недели по приоритетам (must/should/could);
- ежедневное выделение 2-3 ключевых задач дня;
- ограничение контекст-переключений: работа блоками по 60-90 минут.

### 6.3. Концентрация (700 гр)

- режим deep work: отключение лишних уведомлений на время фокус-сессий;
- правило одной задачи: не начинать новую до фиксации текущего результата;
- работа с чеклистами для снижения потерь внимания на рутинных этапах.
