# Лабораторная работа 5-6

## Тема

Изучение возможностей ИТ-системы `Proaction.pro` для оценки персонала.

По заданию нужен доступ команды к демо-системе и прохождение тестов. Полностью завершить работу без логина, пароля и командной оценки нельзя, поэтому ниже подготовлен готовый каркас отчета. После получения доступа нужно только вставить фактические данные и скриншоты.

## 1. Цель работы

Изучить функциональные возможности системы `Proaction.pro` для оценки персонала, рассмотреть применяемые методы оценки и показать пример результатов тестирования.

## 2. Краткое описание системы

`Proaction.pro` можно охарактеризовать как цифровую платформу для оценки кандидатов и сотрудников, ориентированную на автоматизацию тестирования, диагностики компетенций и аналитики результатов. Такие системы используются для первичной оценки при подборе, внутренней оценке персонала, развитии сотрудников и формировании кадровых решений.

## 3. Возможный функционал системы

В отчете можно описать следующие блоки:

- создание и администрирование учетных записей;
- приглашение участников к тестированию;
- выбор тестов и методик;
- запуск оценки и сбор результатов;
- визуализация результатов;
- формирование отчетов;
- использование итогов оценки для HR-решений.

## 4. Методы оценки персонала, которые целесообразно выделить

Если они представлены в системе, в отчете можно описать:

- тесты способностей;
- личностные опросники;
- оценку профессиональных склонностей;
- поведенческие и компетентностные профили;
- сравнительные аналитические отчеты по нескольким участникам.

## 5. Шаблон содержательной части отчета

### 5.1. Назначение системы

Система предназначена для цифровой оценки кандидатов и сотрудников с целью повышения объективности HR-решений. Она позволяет ускорить подбор, стандартизировать оценку и представить результаты в удобной аналитической форме.

### 5.2. Основные функции

- регистрация и настройка участников;
- назначение тестов;
- дистанционное прохождение оценки;
- автоматическая обработка результатов;
- визуальное представление результатов;
- формирование отчетов для HR и руководителей.

### 5.3. Преимущества использования

- сокращение времени на первичную оценку;
- единый цифровой формат результатов;
- удобство сравнения кандидатов;
- возможность удаленного тестирования;
- снижение субъективности оценивания.

### 5.4. Ограничения

- зависимость от корректности выбранной методики;
- необходимость доступа к системе;
- необходимость реальных участников для демонстрации полной логики оценки;
- часть выводов требует профессиональной интерпретации HR-специалистом.

## 6. Что добавить после получения доступа

В итоговую версию нужно вставить:

1. Скриншот главной страницы системы.
2. Скриншот раздела выбора тестов.
3. Скриншот приглашения участника на тестирование.
4. Скриншот примера результатов оценки.
5. Краткое описание того, какой именно тест проходили участники.

## 7. Пример заключения

Система `Proaction.pro` может рассматриваться как эффективный инструмент цифровой оценки персонала. Она позволяет организовать дистанционное тестирование, стандартизировать подход к оценке и получить формализованные результаты, пригодные для анализа. Практическая ценность системы особенно заметна в задачах массового подбора, первичного отбора кандидатов и поддержки кадровых решений.
