# Задание 6.3. Обоснование выбора методов исследования

## Тема ВКР

Разработка интеллектуальной системы ранжирования кандидатов для ИТ-рекрутмента.

## Краткое обоснование выбора методов

Для данной темы выбран комбинированный исследовательский дизайн, так как задача включает и техническую, и организационно-управленческую составляющие.

1. **Анализ научной литературы и нормативных источников** необходим для формирования корректной теоретической базы, терминов и критериев качества.
2. **Контент-анализ вакансий и резюме** позволяет формализовать компетенции и выделить признаки для алгоритмов сопоставления.
3. **Экспертные интервью с рекрутерами и тимлидами** нужны для валидации критериев отбора и учета отраслевой специфики.
4. **Экспериментальное сравнение моделей ранжирования** обеспечивает проверку гипотез об эффективности разработанного решения.
5. **Статистический анализ метрик качества** (`Precision@K`, `Recall`, `F1`, `NDCG`) позволяет объективно оценить результат и сопоставить его с базовым подходом.

Выбранный набор методов обеспечивает и научную обоснованность, и практическую применимость результатов в реальном процессе ИТ-подбора.

## Дополнение: логика выбора методов по этапам ВКР

| Этап ВКР | Метод | Что подтверждаем |
|---|---|---|
| Постановка проблемы | Анализ литературы | Научную релевантность и незакрытые исследовательские вопросы |
| Формализация требований | Контент-анализ вакансий/резюме | Набор признаков и критериев сопоставления |
| Проектирование решения | Экспертные интервью | Практические ожидания HR и технических команд |
| Проверка гипотез | Эксперимент | Преимущество разработанной модели над базовой |
| Интерпретация результата | Статистический анализ | Достоверность полученных улучшений |

## Обоснование методической триангуляции

Использование только одного метода создает риск односторонних выводов: качественные методы дают глубину, но ограничены статистически; количественные дают строгие метрики, но могут терять контекст. Их комбинация повышает достоверность и практическую применимость результата.

## Ожидаемый результат методического блока

1. Формирование проверяемой исследовательской гипотезы.
2. Получение статистически и практически значимых результатов.
3. Подготовка доказательной базы для раздела ВКР «Методы исследования».
