# Инвариантное задание 5. Кодексы этики

## Рассмотренные кодексы

- ACM Code of Ethics;
- IEEE Code of Ethics;
- внутренние корпоративные кодексы ИТ-компаний.

## Общие принципы

1. Не навреди пользователю и обществу.
2. Действуй честно и прозрачно.
3. Уважай права автора и право на приватность.
4. Повышай качество и надежность создаваемых систем.
5. Непрерывно развивай профессиональные компетенции.

## Практическое применение

При проектировании цифрового сервиса необходимо заранее оценивать этические риски и закладывать механизмы защиты: контроль доступа, аудит действий, документирование решений.
