# ВСР 7. Обзор курсов ДО по ИТ

## Цель

Сформировать обзор дистанционных ИТ-курсов для самостоятельного профессионального развития.

## Аннотированный список курсов

| Платформа | Курс | Формат | Что дает |
|---|---|---|---|
| Stepik | Введение в программирование | Видео + практика | Системная база по алгоритмическому мышлению |
| Coursera | Foundations of Project Management | Видео + тесты | Базовые подходы к управлению проектами |
| Яндекс Практикум | Веб-разработчик | Тренажеры + проекты | Практические навыки frontend/backend |
| Hexlet | Основы JavaScript | Практика в браузере | Базовые навыки JS для веб-разработки |
| OpenEdu | Цифровая грамотность и ИКТ | MOOC | Широкий контекст цифрового общества и инструментов |

## Критерии выбора курсов

1. Наличие практических заданий и проектного трека.
2. Обратная связь и возможность ревью работ.
3. Понятная траектория от базовых к продвинутым темам.
4. Актуальность программы относительно рынка труда.

## Вывод

Оптимальная стратегия — комбинировать фундаментальные и прикладные курсы: теория дает системность, практика — готовность к реальным рабочим задачам.
