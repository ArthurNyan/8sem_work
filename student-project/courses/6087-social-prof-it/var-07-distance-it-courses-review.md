# Вариативное задание 7. Сравнительный обзор курсов ДО по ИТ

## Цель обзора

Подобрать дистанционные курсы под тему ВКР:  
`Разработка системы управления контентом в сфере маркетинга на базе Strapi и Astro`  
и под карьерный трек: `middle frontend -> fullstack -> архитектор`.

## Критерии сравнения

1. Релевантность стеку `Strapi`, `Astro`, `TypeScript`, `Node.js`.
2. Практическая направленность (pet-проекты, capstone, портфолио).
3. Качество обратной связи и менторства.
4. Покрытие архитектурных тем (API-дизайн, CI/CD, масштабирование, безопасность).
5. Применимость к задачам маркетингового контента (CMS, SEO, персонализация, аналитика).

## Сравнительный обзор

| Площадка/курс | Сильные стороны | Ограничения | Польза для ВКР и карьеры |
|---|---|---|---|
| Strapi Learn + документация | Точный вход в `Strapi` v4/v5, RBAC, Content Types, REST/GraphQL | Меньше системного пути "с нуля до production" | База для backend-части CMS и контент-моделирования |
| Astro Docs + Astro Learn | Официальный подход к архитектуре Astro, islands, производительность и SEO | Нужна самостоятельная практика на реальном проекте | Основа frontend-части маркетинговой CMS |
| Fullstack Open | Сильная инженерная база по fullstack, API и DevOps-практикам | Меньше фокуса на Strapi/Astro напрямую | Усиление fullstack-мышления и качества кода |
| Node.js/Backend архитектурные курсы (Udemy/Coursera) | Проектирование сервисов, auth, кэширование, наблюдаемость | Качество сильно зависит от конкретного автора курса | Шаг к роли архитектора и системному проектированию |
| Курсы по Product/Software Architecture | Архитектурные паттерны, ADR, NFR, масштабирование | Требуют опыта внедрения для полного эффекта | Переход от уровня реализации к архитектурным решениям |

## Вывод по выбору

Оптимальная стратегия - не один курс, а связка:
1. `Strapi Learn` + документация.
2. `Astro Learn` + практический проект.
3. Один системный fullstack/архитектурный курс с упором на backend и проектирование.

Такой набор напрямую поддерживает ВКР, усиливает компетенции fullstack-разработчика и формирует основу для перехода к роли архитектора.
