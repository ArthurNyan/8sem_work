# Вариативное задание 7. Сравнительный обзор курсов ДО по ИТ

## Критерии сравнения

1. Актуальность программы.
2. Наличие практических заданий.
3. Качество обратной связи.
4. Подтверждение результатов (сертификат, проект).

## Результат сравнения

Для развития карьерного профиля наиболее эффективны курсы, где есть проект в портфолио и регулярная проверка работ, а не только видеолекции и тесты.
