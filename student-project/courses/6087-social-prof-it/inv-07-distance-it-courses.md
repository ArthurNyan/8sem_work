# Инвариантное задание 7. Обзор курсов ДО по ИТ

## Цель обзора

Сопоставить дистанционные ИТ-курсы по содержанию, формату и практической ценности для профессионального развития.

## Краткий обзор платформ

| Платформа | Сильные стороны | Ограничения |
|---|---|---|
| Stepik | Практико-ориентированные курсы, русскоязычный контент | Разный уровень методической проработки |
| Coursera | Университетские программы, сертификаты | Часть контента платная |
| OpenEdu | Академические дисциплины, структура ВО | Меньше гибкости по темпу |

## Вывод

Оптимальная стратегия: сочетать фундаментальные курсы (OpenEdu/Coursera) с прикладными практикумами (Stepik).
