# ВСР 3. Информационно-поисковые системы (ИПС)

## Цель

Сравнить возможности ИПС для учебной и исследовательской работы по выбранной теме ВКР.

## Сравнительная таблица ИПС

| ИПС | Тип контента | Сильные стороны | Ограничения |
|---|---|---|---|
| Google Scholar | Научные публикации | Быстрый поиск статей, цитируемость | Не всегда полный доступ к текстам |
| eLIBRARY | Российские научные источники | Большой массив русскоязычных публикаций | Нужна аккуратная фильтрация качества |
| CyberLeninka | Открытые статьи | Доступность полных текстов | Неравномерное качество материалов |
| Scopus / WoS (обзорно) | Международные базы | Высокие стандарты отбора | Ограниченный доступ без подписки |
| Яндекс / Google | Общий веб-поиск | Быстрый обзор темы и трендов | Много нерецензируемых материалов |

## Рекомендованная стратегия поиска

1. Сначала формулировать запрос через ключевые термины и синонимы.
2. Делить источники на научные, аналитические и прикладные.
3. Проверять релевантность по дате, автору, цитируемости, контексту.
4. Для финальной базы использовать только проверяемые источники.

## Аннотированный мини-список ресурсов по теме ВКР

1. `Google Scholar` — обзор актуальных исследований по применению ИИ в разработке и управлении проектами.
2. `eLIBRARY` — русскоязычные публикации по цифровой трансформации и ИТ-компетенциям.
3. `CyberLeninka` — открытые статьи для формирования теоретического раздела.
4. `Habr` (как дополнительный прикладной источник) — кейсы индустрии, используемые с критической проверкой.

## Вывод

Наиболее качественный результат дает комбинированный подход: академические ИПС для доказательной базы и прикладные ресурсы для описания современных практик.
