# Практическая работа 5. Профессиональная этика

## Основные положения

Профессиональная этика ИТ-специалиста включает:

- ответственность за последствия цифровых решений;
- недопустимость дискриминации в алгоритмах;
- прозрачность и объяснимость решений;
- уважение приватности пользователя;
- корректную работу с интеллектуальной собственностью.

## Этические риски

1. Неэтичный сбор и использование персональных данных.
2. Алгоритмические искажения (bias).
3. Манипулятивные интерфейсы.
4. Недостоверная генерация контента ИИ-моделями.
