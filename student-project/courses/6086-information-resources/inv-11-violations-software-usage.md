# Инвариантное задание 11. Нарушения в использовании программных продуктов

## Таблица нарушений

| Нарушение | Ответственность | Принимаемые меры |
|---|---|---|
| Использование нелицензионного ПО | Административная, гражданская | Штраф, компенсация, запрет использования |
| Незаконное копирование и распространение | Гражданская, административная, в ряде случаев уголовная | Блокировка, взыскание убытков |
| Нарушение условий лицензии | Гражданская | Расторжение лицензии, компенсация |
| Внедрение вредоносного ПО | Уголовная | Уголовное наказание, ограничение свободы |
| Несанкционированный доступ к данным | Уголовная | Наказание по нормам о неправомерном доступе |

## Вывод

Соблюдение авторских и пользовательских прав в сфере ПО — обязательное условие законной и профессиональной деятельности в ИТ.
