# Вариативное задание 2. Работа с открытыми фондами военных архивов

## Цель

Показать возможности открытых архивных ресурсов по тематике Великой Отечественной войны и военной истории.

## Примеры открытых архивных платформ

| Ресурс | URL | Что можно найти |
|---|---|---|
| Память народа | https://pamyat-naroda.ru | Боевые документы, карты операций, сведения о награждениях |
| Подвиг народа | https://podvignaroda.ru | Наградные документы и архивные листы |
| ОБД Мемориал | https://obd-memorial.ru | Данные о погибших и пропавших без вести |
| Российский государственный военный архив | https://rgvarchive.ru | Информация о фондах и архивных коллекциях |

## Практические шаги работы с архивом

1. Сформулировать объект поиска (человек, подразделение, операция, период).
2. Выделить ключевые параметры запроса (дата, место, фамилия, номер части).
3. Сопоставить данные из 2-3 источников для верификации.
4. Зафиксировать найденные документы и источники в аннотированном списке.

## Вывод

Открытые военные архивы позволяют проводить исследовательскую работу на основе первичных источников. При этом важна аккуратность проверки и корректное документирование найденных материалов.
