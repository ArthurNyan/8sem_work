# Вариативное задание 5. Электронные ресурсы

## Выбранный вариант выполнения

Выполнен вариант анализа конкретного электронного ресурса:
**Научная электронная библиотека eLIBRARY.RU**.

## Карточка ресурса

| Параметр | Описание |
|---|---|
| Название | eLIBRARY.RU |
| URL | https://www.elibrary.ru |
| Тип | Научный электронный информационный ресурс |
| Назначение | Поиск публикаций, анализ цитируемости, подбор научных источников |
| Основные пользователи | Студенты, преподаватели, исследователи |

## Функциональные возможности

1. Поиск по ключевым словам, автору, журналу, тематике.
2. Отбор публикаций по научному профилю.
3. Работа с библиографическими данными.
4. Анализ публикационной активности и цитируемости.

## Оценка качества ресурса

| Критерий | Оценка | Комментарий |
|---|---|---|
| Актуальность | Высокая | регулярно обновляемые научные записи |
| Достоверность | Высокая | формальная научная публикационная среда |
| Полнота | Высокая | большой объем русскоязычных материалов |
| Удобство интерфейса | Средняя | требует навыка расширенного поиска |

## Практическое применение в учебной работе

1. Подбор теоретической базы для рефератов и ВКР.
2. Проверка корректности библиографического описания.
3. Поиск авторитетных источников по теме информатики и ИТ.

## План слайдов (если сдача в формате презентации)

1. Что такое электронные ресурсы и их роль в учебе.
2. Выбор ресурса eLIBRARY.RU.
3. Основные функции и интерфейс.
4. Пример поискового запроса по теме ВКР.
5. Оценка качества информации.
6. Плюсы и ограничения.
7. Вывод.

## Вывод

eLIBRARY.RU является базовым электронным ресурсом для подготовки академических работ благодаря сочетанию масштабной научной базы и инструментов библиографического поиска.
