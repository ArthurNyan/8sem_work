# ВСР 11. Регистрация программного продукта и права пользователя

## Задание 11.1. Список действий для регистрации ПО

1. Определить правообладателя и форму владения правами.
2. Подготовить техническое описание и идентификационные сведения.
3. Оформить заявление и комплект сопровождающих документов.
4. Подать документы в регистрирующий орган.
5. Получить подтверждение регистрации.
6. Зафиксировать данные в корпоративной документации.

## Задание 11.2. Права пользователя программного продукта

1. Право использования продукта в пределах лицензии.
2. Право на получение документации и информации об ограничениях.
3. Право на получение обновлений (при наличии по договору).
4. Право на техническую поддержку.
5. Право на защиту персональных данных.
6. Право требовать соблюдения условий поставки и качества.

## Вывод

Корректная регламентация прав пользователя и процедуры регистрации ПО необходимы для правовой устойчивости ИТ-проектов и безопасности всех участников использования программного продукта.
