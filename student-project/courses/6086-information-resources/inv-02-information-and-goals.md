# Инвариантное задание 2. Информация и цели её получения

## Таблица возможных нарушений в использовании программных продуктов

| Нарушение | Ответственность | Принимаемые меры |
|---|---|---|
| Использование нелицензионного ПО | Гражданская, административная | Штрафы, компенсация правообладателю, удаление ПО |
| Незаконное распространение копий | Гражданская, административная, в отдельных случаях уголовная | Блокировка распространения, взыскание убытков |
| Обход технических средств защиты | Административная/уголовная (по составу) | Изъятие материалов, санкции по решению суда |
| Модификация ПО без разрешения правообладателя | Гражданская | Прекращение использования, компенсация |
| Использование вредоносного ПО | Уголовная | Уголовное преследование, ограничение свободы, иные санкции |
| Незаконный доступ к данным через ПО | Уголовная | Ответственность по статьям о неправомерном доступе |

## Комментарий

Информационные продукты существуют в правовом поле. Поэтому цель работы с информацией — получить полезный результат без нарушения авторских, имущественных и пользовательских прав.

## Вывод

Правовая грамотность в обращении с ПО и информацией — обязательная часть компетенций современного ИТ-специалиста.
