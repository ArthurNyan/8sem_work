# Инвариантное задание 3. Мировые информационные агентства

## Обзор мировых агентств

| Агентство | Регион | Ключевые направления | Ценность для исследовательской работы |
|---|---|---|---|
| Reuters | Глобальный | Финансы, геополитика, технологии | Оперативная и верифицируемая международная повестка |
| AP | США/международное | Новости, фото, видео | Качественная фактологическая база |
| AFP | Европа/мир | Политика, международные события | Широкое освещение европейской и глобальной тематики |
| DPA | Германия/ЕС | Европейская повестка, экономика | Полезно для сопоставления региональных акцентов |
| ТАСС | Россия/международное | Государственная и общественно-политическая повестка | Контекст российской информационной картины |

## Методика использования агентств

1. Сравнивать новость минимум по двум независимым агентствам.
2. Выделять факт и оценочный комментарий отдельно.
3. Проверять первоисточник, если агентство ссылается на заявление/документ.
4. Фиксировать время публикации для понимания динамики сюжета.

## Вывод

Мировые информационные агентства формируют базовую инфраструктуру новостного обмена. Их сравнительный анализ помогает снижать риск информационного искажения.
