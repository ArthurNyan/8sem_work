# Практическая работа 8. Архивы научных журналов

## Тема

Поиск профильных публикаций в архивах научных журналов для включения в исследовательскую работу.

## Результат поиска (пример)

| Журнал | Тема статьи (пример) | Почему релевантно |
|---|---|---|
| Информатика и образование | Цифровая образовательная среда | Связь с информационными ресурсами и их применением |
| Программные продукты и системы | Управление ИТ-системами | Практический аспект информационных услуг |
| Научно-техническая информация | Поиск и обработка научных данных | Методическая база для работы с информацией |

## Рекомендации по работе с архивами

1. Использовать точные тематические ключевые слова.
2. Проверять `Abstract` на соответствие исследовательской задаче.
3. Сразу фиксировать библиографическое описание.
4. Отбирать источники по новизне и научной значимости.

## Вывод

Архивы научных журналов — важный источник проверяемых материалов для учебных и научных работ, особенно на этапе формирования теоретической базы.
