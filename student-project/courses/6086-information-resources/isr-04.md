# ИСР 4

## Выполнение задания 1 (по практическому занятию 4)

Подготовлен сравнительный анализ электронных информационных ресурсов для учебно-исследовательской деятельности.

## Таблица анализа ресурсов

| Ресурс | Тип | Ключевая функция | Плюсы | Ограничения |
|---|---|---|---|---|
| eLIBRARY | Научная база | Поиск научных публикаций | Большой русскоязычный массив статей | Не весь контент в open access |
| КиберЛенинка | Научный open access | Доступ к полнотекстовым статьям | Быстрый доступ без подписки | Разный уровень качества журналов |
| РГБ (электронный каталог) | Библиотечный ресурс | Поиск книг и диссертаций | Высокая библиографическая ценность | Часть источников недоступна онлайн |
| РИНЦ | Библиометрический ресурс | Анализ публикационной активности | Удобно для оценки цитируемости | Нужны навыки интерпретации метрик |
| ТАСС | Новостной источник | Оперативная повестка | Актуальность событий | Не заменяет научные источники |

## Практические критерии выбора источника

1. Соответствие теме исследования.
2. Достоверность и официальный статус.
3. Возможность корректного цитирования.
4. Актуальность публикации.
5. Наличие полного текста.

## Вывод

Для устойчивого качества работы необходима комбинация источников: научные базы для аргументации, библиотечные каталоги для библиографии и новостные агентства для актуального контекста.
