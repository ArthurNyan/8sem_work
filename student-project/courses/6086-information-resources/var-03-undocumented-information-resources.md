# Вариативное задание 3. Недокументированные информационные ресурсы

## Понятие

Недокументированные информационные ресурсы — это данные и знания, которые не оформлены как официальный документ, но используются в профессиональной деятельности (устные экспертные комментарии, интервью, наблюдения, профессиональные сообщества).

## Примеры недокументированных ресурсов

1. Экспертные интервью и комментарии специалистов.
2. Профессиональные дискуссии в отраслевых сообществах.
3. Наблюдения за процессами в организации.
4. Кейсы и практики, не оформленные в виде публикаций.

## Риски использования

- субъективность источника;
- неполнота и контекстная ограниченность;
- сложность верификации;
- риск некорректного цитирования.

## Правила корректной работы

1. Сопоставлять недокументированные данные с документированными источниками.
2. Явно фиксировать статус информации как экспертной оценки.
3. Не использовать непроверенные данные в качестве единственного доказательства.
4. Документировать контекст получения информации (дата, источник, условия).

## Вывод

Недокументированные ресурсы полезны для выявления практических нюансов, но требуют критической проверки и аккуратного включения в аналитические материалы.
