# ИСР 11. Нарушения в области использования программных продуктов

## Расширенная таблица

| Нарушение | Ответственность | Наказание / меры |
|---|---|---|
| Использование пиратского ПО в организации | Административная, гражданская | Штраф, компенсация, изъятие нелицензионных копий |
| Распространение нелицензионных сборок | Гражданская, административная, уголовная | Блокировка каналов распространения, взыскание убытков |
| Нарушение условий корпоративной лицензии | Гражданская | Расторжение лицензионного договора, финансовые санкции |
| Создание и распространение вредоносного ПО | Уголовная | Уголовная ответственность и санкции по приговору суда |
| Взлом программ и обход защиты | Уголовная/административная | Штрафы, иные меры в зависимости от состава нарушения |

## Практические меры предотвращения

1. Лицензионный аудит установленного ПО.
2. Централизованное управление доступом и установкой программ.
3. Обучение сотрудников цифровой и правовой грамотности.
4. Регулярные проверки информационной безопасности.

## Вывод

Профилактика нарушений в работе с ПО строится на сочетании организационных, технических и правовых мер.
