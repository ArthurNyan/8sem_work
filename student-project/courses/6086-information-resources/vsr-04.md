# ВСР 4

## Тема

Сравнение способов поиска информации для учебного проекта.

## Сравнение подходов

| Подход | Преимущества | Ограничения |
|---|---|---|
| Общий web-поиск | Скорость, широкий охват | Высокий информационный шум |
| Научные базы | Качество и цитируемость | Не всегда свободный доступ |
| Библиотечные каталоги | Структурированность и проверяемость | Требует больше времени на поиск |

## Рекомендованный алгоритм

1. Сформулировать тему и ключевые слова.
2. Выполнить первичный обзор в web-поиске.
3. Углубить материал через научные базы.
4. Подтвердить данные по официальным/библиотечным источникам.

## Вывод

Результативный поиск строится на сочетании скорости и качества: быстрый обзор + верификация через научные и официальные источники.
