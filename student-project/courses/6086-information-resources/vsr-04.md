# ВСР 4

## Выполнение задания 2 (по практическому занятию 4)

Сформирован алгоритм углубленного поиска и верификации информации для подготовки учебной работы.

## Сравнение стратегий поиска

| Стратегия | Когда применять | Результат |
|---|---|---|
| Быстрый web-поиск | Стартовое знакомство с темой | Набор первичных ориентиров |
| Поиск по научным базам | Подбор доказательной базы | Статьи, обзоры, рецензируемые материалы |
| Поиск по библиотечным каталогам | Формирование списка литературы | Корректные библиографические записи |
| Поиск по официальным ресурсам | Нормативные и статистические данные | Верифицированные документы |

## Пошаговый рабочий алгоритм

1. Определить исследовательский вопрос и ключевые термины.
2. Провести первичный сбор источников через общий поиск.
3. Отобрать надежные публикации в научных базах.
4. Проверить библиографические данные по каталогам библиотек.
5. Провести финальную верификацию фактов через официальные порталы.
6. Зафиксировать источники в едином списке литературы.

## Типовые ошибки и способы их избежать

| Ошибка | Риск | Решение |
|---|---|---|
| Использование только web-источников | Низкая академическая достоверность | Добавлять рецензируемые публикации |
| Отсутствие даты обращения | Потеря воспроизводимости | Фиксировать дату и URL |
| Смешение фактов и мнений | Снижение качества аргументации | Разделять данные и интерпретации |

## Вывод

Качественный поиск строится на многоуровневой модели: обзор, научная проверка, официальная верификация, корректное библиографическое оформление.
