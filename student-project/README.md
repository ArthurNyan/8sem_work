# Учебный проект

В этой папке собраны материалы и черновики решений по курсам, выгруженным из Moodle.

## Структура

- `courses/6086-information-resources` — курс "Информационные ресурсы"
- `courses/6087-social-prof-it` — курс "Социальные и профессиональные вопросы информатики и ИТ"
- `courses/16582-it-recruitment` — курс "IT-рекрутмент"
- `courses/16583-hr-evaluation-tech` — курс "Информационные технологии оценки персонала"
- `courses/31896-research-project` — курс "Научно-исследовательский проект"

## Статус

На текущем этапе подготовлены:

- готовые текстовые решения по нескольким курсам;
- материалы в формате Markdown, пригодные для конвертации в PDF;
- часть работ в формате, близком к итоговой сдаче (включая проектные отчеты и планы).

## Что ещё потребуется перед сдачей

- при необходимости перенести отдельные работы в `.docx` или `.pptx` (если преподаватель требует именно эти форматы);
- добавить скриншоты/вложения в заданиях, где это указано отдельным пунктом;
- проверить требования преподавателя к структуре титульного листа и объему.
