# Учебный проект

В этой папке собраны материалы и черновики решений по курсам, выгруженным из Moodle.

## Структура

- `courses/16582-it-recruitment` — курс "IT-рекрутмент"
- `courses/16583-hr-evaluation-tech` — курс "Информационные технологии оценки персонала"

## Статус

На текущем этапе подготовлены:

- карта заданий по двум курсам;
- черновики и полуготовые решения по текстовым заданиям;
- шаблоны для работ, где нужны скриншоты, демо-доступы или финальная упаковка в `.docx/.pptx`.

## Что ещё потребуется перед сдачей

- при необходимости перенести готовые тексты в `.docx` или `.pptx`;
- добавить скриншоты интерфейсов и демо-систем там, где это явно требуется заданием;
- проверить, не было ли у преподавателя дополнительных требований по объему и оформлению.
